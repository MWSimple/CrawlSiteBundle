CrawlSiteBundle
===============

## Installation

### Using composer

Add following lines to your `composer.json` file:

### Symfony 2.3.*

```json
"require": {
    ...
    "mwsimple/crawl-site": "2.3.*@dev"
}
```

Execute:

```cli
php composer.phar update "mwsimple/crawl-site"
```

### Crawler Sites

```cli
php app/console mwsimple:crawlsite
```
